Overview

  • Two types of merges:
    1. fast-forward
    2. true merge
      • merge commits
  • Fast-forward options
  • Squash merging

Fast-forward Merge

Default behavior when merging a non-divergent branch.

Fast-forward Merge

git merge feature-branch ➜

Fast-forward Merge (continued)

True Merge

Required when merging a divergent branch.

True Merge

git merge feature-branch ➜

True Merge (continued)

Fast Forward Options

git merge --ff <branch>

(Default) Fast-forward when possible, otherwise create a merge commit.

git merge --no-ff <branch>

Always create a merge commit.

git merge --ff-only <branch>

Fast-forward when possible, otherwise exit.

Squash Merging

git merge --squash <branch>

Squash all commits from branch into a new commit upon merging.
